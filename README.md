# ianstagram


Projeto realizado na disciplina Práticas de Programação para segunda unidade avaliativa

## Contributing

1. Faça o _fork_ do projeto (<https://github.com/ianmichaelxp/ianstagram/fork>)
2. Crie uma _branch_ para sua modificação (`git checkout -b feature/fooBar`)
3. Faça o _commit_ (`git commit -am 'Add some fooBar'`)
4. _Push_ (`git push origin feature/fooBar`)
5. Crie um novo _Pull Request_

## COMANDOS FRONT-END

1. Na pasta front-end faça `npm install` para atualizar as dependências
2. Em seguida `ng serve --open`

## COMANDOS BACK-END

1. Na pasta back-end faça `npm install` para atualizar as dependências
2. Em seguida `npm start`

## Requisitos

1. Node.js 8.X
2. Angular CLI versão 6.X
